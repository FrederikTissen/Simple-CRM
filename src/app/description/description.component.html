Project description

Welcome to my "Simple-CRM".

With this project I want to show you some of my programming skills with Angular.
CRM systems are complex and only make sense in reality if they are filled with a lot of data and linked to databases.
Of course I could not realize this complexity here. Here you can create, edit and delete user data and save them in a backend. Also you can see the implementation of a login, charts and a simple calendar.

Frontend: Angular, Angular Material, Chart.js
Backend: Firebase
